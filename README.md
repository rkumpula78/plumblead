# PlumbLead.ai

**AI-powered instant quotes + speed-to-lead for plumbing contractors**

- Domain: plumblead.ai
- Started: 2026-03-26
- Status: MVP Build

## Product Overview

PlumbLead helps plumbing contractors convert more leads into booked jobs by:
1. **Instant Quote Tool** — embeds on their website, gives homeowners a ballpark price in seconds
2. **Speed-to-Lead AI** — responds to every inbound lead (form, call, chat) within 60 seconds via text/email
3. **AI Chatbot** — qualifies leads 24/7, answers common questions, books appointments
4. **CRM Integration** — pushes qualified leads directly into ServiceTitan or Housecall Pro

## Target Customer

- Residential plumbing companies in Phoenix/Arizona (initial)
- 2-20 trucks / $500K-$5M revenue
- Already spending on Google/LSA ads but losing leads to slow follow-up
- Using ServiceTitan or Housecall Pro

## Pricing

- **Setup:** $2,500 (one-time)
- **Monthly:** $500/mo (includes AI, hosting, support, optimization)
- **ROI pitch:** One extra water heater install per month ($1,500-3,500) pays for the entire system
